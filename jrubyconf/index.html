REST

Taking these concepts to the next level, we have "Rucene", a sample application which uses Sinatra in JRuby to provide a simple REST API for Lucene, and Sinatra in Ruby to provide the user-facing interface.

Look at the code for rucene.

DRb

When you need to use JRuby, but need much tighter integration between it and your main Ruby application than a REST API or separate process, you can use DRb. We looked at how Akephalos uses DRb to communicate between Capybara (an integration testing framework) and HTMLUnit (a headless browser build with Java).

Look at the code for akephalos.
